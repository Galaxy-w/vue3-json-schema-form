# vue3-json-schema-form

a form generator based on jsonschema for vue3

## Project setup

```bash
npm install
npm run serve
npm run build
npm run test:unit
npm run lint
```

## git

```bash
# 查看远程分支
git granch -r
# 查看所有分支
git branch -a

# 查看第3章的分支
gb -r | grep origin/3
```
